# 성적 분포 분석 시스템

> 입력된 점수 데이터를 기반으로 기초 통계량과 분포를 분석하여 성적 분포를 시각적으로 보여준다.

---

### 풀려는 문제 또는 만들려는 이유

본 시스템은 학부생들의 성적 데이터를 보다 직관적으로 표현하기 위한 시스템이다. 기존의 평균과 같은 기초 통계값만으로는 전체 성적 분포를 파악하기 어려운 한계가 있다. 따라서 최소값, 사분위수, 중앙값, 최대값 등의 통계량과 히스토그램(Histogram), 박스플롯(Boxplot)을 통해 데이터를 시각적으로 표현하고자 한다. 또한 사용자 입력 또는 파일 데이터를 기반으로 분석 결과를 한 번에 확인할 수 있는 기능을 제공하는 것을 목표로 한다.

---

### 핵심 기능 3가지

- **기초 통계량** 구하기 : 데이터를 오름차순으로 정렬하여 통계량을 구하고 IQR을 활용해 이상치 판단
- **Histogram** 구현 : 사용자가 원하는 구간의 크기를 입력받아, 각 구간에 속하는 데이터의 빈도를 계산하여 시각화
- **Boxplot** 구현 : 중앙값과 사분위수를 기반으로 데이터 분포를 Boxplot 형태로 시각화

---

### 예상 사용자 : 교수자 및 학부생 등 성적 데이터를 분석하고자 하는 사람

---

### 결과물의 형태 : CLI

---

### 사용할 Python 개념

- 리스트, 딕셔너리 : 점수 데이터 저장 및 구간별 빈도수 파악 
- 조건문, 반복문 
- 함수 : 기능별 함수 처리
- 파일 입출력 : 성적 데이터 읽어오기
- 예외 처리 : 파일이 없거나 숫자가 아닌 값이 입력되었을 때의 오류 방지
