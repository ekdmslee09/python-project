# 성적 분포 분석 시스템

> 입력된 점수 데이터를 기반으로 기초 통계량을 계산하고 성적 분포를 시각적으로 보여준다.

---

### 풀려는 문제 또는 만들려는 이유

본 시스템은 학부생들의 성적 데이터를 보다 직관적으로 표현하기 위한 시스템이다. 기존의 평균과 같은 기초 통계량만으로는 전체 성적 분포를 파악하기 어려운 한계가 있다. 따라서 최소값, 사분위수, 중앙값, 최대값 등의 통계량과 히스토그램(Histogram), 박스플롯(Boxplot)을 통해 데이터를 시각적으로 표현하고자 한다. 또한 사용자 입력을 기반으로 분석 결과를 한 번에 확인할 수 있는 기능을 제공하는 것을 목표로 한다.

---

### 핵심 기능 3가지

- **기초 통계량** 구하기 : 데이터를 오름차순으로 정렬한 뒤 통계량을 계산.
- **Histogram** 구현 : 20점 단위로 5개의 구간으로 나눈 뒤 각 구간에 속하는 데이터의 빈도를 계산하여 시각화
- **Boxplot** 구현 : 최소값, 사분위수, 중앙값, 최대값을 기반으로 데이터 분포를 Boxplot 형태로 시각화

---

### 예상 사용자 : 교수자 및 학부생 등 성적 데이터를 분석하고자 하는 사람

---

### 결과물의 형태 : CLI

---

### 사용할 Python 개념

- 리스트 : 점수 데이터 저장
- 딕셔너리 : 기초 통계량 저장 
- 조건문 : 점수 범위 검사 및 구간 분류
- 반복문 : 데이터 입력 및 그래프 출력
- 함수 : 기능별로 함수를 분리하여 구현
- 예외 처리 : 숫자가 아니거나 범위를 벗어난 점수 입력에 대한 오류 처리
