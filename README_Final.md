# 성적 분포 분석 시스템
 
## 무엇을 하는 프로그램인가요?
사용자로부터 학생들의 점수 데이터를 입력받아 기초 통계량(평균, 중앙값, 사분위수 등)을 산출하고, 이를 히스토그램과 박스플롯으로 시각화하여 데이터의 분포와 특성을 직관적으로 파악할 수 있게 돕는 CLI 기반의 분석 시스템입니다. 
 
## 실행 방법
1. python 3.10 이상 설치
2. python Final.py 실행
 
## 사용된 Python 개념
- 리스트, 딕셔너리, 조건문, 반복문, 함수, 예외 처리
